## 摘要

多智能体协作模式将多个专业智能体组织成团队，通过明确分工、通信协议和协调机制，共同完成单一智能体无法独立处理的复杂任务。

## 详情

### 模式概述

多智能体协作的核心是**分工与协调**——将复杂任务拆分为多个子任务，分配给具有不同专业能力的智能体，通过结构化的通信机制协调工作 。

**协作结构类型**：
- **链式**：A → B → C，前一个智能体的输出是后一个的输入
- **星式**：中心智能体调度多个周边智能体
- **网状**：多个智能体自由通信，无固定中心

### 关系与通信结构

书中深入探讨了智能体间的关系类型和通信协议 ：
- **层级关系**：主管智能体分配任务给子智能体
- **对等关系**：智能体间平等协商
- **通信协议**：结构化消息格式确保信息传递的准确性

### 实践应用

- **研究团队**：文献检索智能体 + 数据分析智能体 + 报告撰写智能体
- **软件开发**：需求分析智能体 + 代码生成智能体 + 测试智能体
- **内容创作**：策划智能体 + 写作智能体 + 审校智能体

### 关键要点

- 多智能体协作适合**复杂度高、需要多领域专业知识**的任务
- 通信开销是主要瓶颈——智能体间通信越多，总延迟越大
- 需要设计明确的**角色边界**和**冲突解决机制**
