## 摘要

路由模式是一种智能分发机制——根据输入内容的特征，自动将请求分类并路由到最合适的处理路径或专家智能体，实现"让对的模型做对的事"。

## 详情

### 模式概述

路由模式的核心是**分类与分发**——先用一个分类器（Classifier）判断输入属于哪个类别，然后将输入路由到对应的专门处理链或专家模型 。

**工作流程**：
```
用户输入 → 分类器 → 类别 A → 专家链 A
                   → 类别 B → 专家链 B
                   → 类别 C → 专家链 C
```

### 实践应用与场景

- **客服系统**：技术问题路由到技术专家，账单问题路由到财务专家
- **内容审核**：不同类别的违规内容路由到不同的审核策略
- **多语言处理**：根据语言类型路由到对应语言优化的模型

### 关键要点

- 路由的准确性取决于**分类器的质量**
- 分类器可以用小模型（成本低）或大模型（准确率高）
- 路由适合**类别明确、互斥性强**的场景

### 代码实现

书中提供了 LangChain 和 Google ADK 两种实现方式 ：
- **LangChain**：使用 LLM 作为路由器，通过结构化输出决定路由
- **Google ADK**：使用内置的路由组件
