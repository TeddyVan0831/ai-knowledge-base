# 提示链（Prompt Chaining）

> **类型**: concept
> **创建时间**: 2026-05-31
> **最后更新**: 2026-05-31
> **来源**: 

## 摘要

提示链（Prompt Chaining）是最基础的智能体设计模式——将复杂任务分解为多个步骤，每一步的输出作为下一步的输入，通过串联提示词实现复杂工作流的自动化。

## 详情

### 模式概述

提示链的核心思想是**分而治之**——将一个大任务拆分成若干小步骤，每个步骤由 LLM 独立处理，上一步的输出直接作为下一步的输入 。

**与单步提示的区别**：
- 单步提示：一个 prompt 要求 LLM 完成所有工作，容易遗漏细节或产生幻觉
- 提示链：多步骤管道，每步专注一个子任务，提高准确性和可控性

### 实践应用与场景

- **内容生成流水线**：大纲生成 → 逐段撰写 → 编辑润色 → 格式调整
- **数据分析流程**：数据清洗 → 特征提取 → 模型推理 → 结果解释
- **代码生成**：需求理解 → 架构设计 → 代码生成 → 测试编写

### 关键要点

- 提示链适合**确定性高、步骤清晰**的任务
- 每步之间可以插入**人工审核**（Human-in-the-Loop）
- 链的长度需要平衡：太长增加延迟，太短降低质量

### 与相关模式的关系

- 提示链是**路由模式**的前置基础（路由决定走哪条链）
- 提示链可与**并行化**组合（多条链并行执行）
- 提示链可与**反思模式**组合（在链中加入自我校验步骤）
