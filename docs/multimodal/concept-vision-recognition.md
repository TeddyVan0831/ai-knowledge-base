# 视觉识别技术

> **类型**: concept
> **创建时间**: 2026-06-02
> **最后更新**: 2026-06-02
> **来源**: 

## 摘要

视觉识别技术涵盖从卷积神经网络（CNN）到 YOLO 目标检测和 UNet 图像分割的完整技术栈。核心概念包括尺度重要性、特征提取、多尺度融合、残差连接和跳跃连接。

## 详情

### 核心概念

**尺度（Scale）**：不同尺度下模型提取的特征不同——近处提取纹理/细节，远处提取结构/形状 。

**特征与向量**：CNN 对每个区域提取固定长度向量（如 128 维、1024 维），所有向量组成特征图 。

**CNN 与框架的关系**：CNN 只是基础元素，需要套上框架（如 YOLO、UNet）才能实现复杂任务，类似经典神经网络需要套上 Transformer 框架 。

### YOLO 目标检测

**由来**：You Only Look Once，核心优势是将两阶段模型的速度大幅提升，实现实时计算（≥10 帧/秒）。

**工作原理**：
1. 将图片划分为 S×S 网格（YOLOv1: 7×7=49）
2. 每个网格判断是否有物体中心点落在其中 
3. 每个网格输出 B×5 + C 个数字（坐标 + 自信度 + 类别概率）

**训练五步法**：定义场景 → 定义输入输出 → 数据标注 → 定义总误差 → 训练模型 。

**多尺度特征融合**：解决 CNN 池化操作导致小物体丢失的问题——将 13×13、26×26、52×52 的特征图上采样后融合，实现大/中/小物体同时识别 。

### UNet 图像分割

**架构**：U 型结构，左侧编码器（特征提取）→ 瓶颈层 → 右侧解码器（上采样还原），通过跳跃连接（Skip Connection）融合编码器特征 。

**核心机制——跳跃连接**：
1. **裁剪**：左侧特征图比右侧稍大，从中心裁剪对齐 
2. **拼接（Concat）**：对应坐标位置的两个向量拼接，通道数翻倍 
3. **融合**：通过卷积层还原到目标维度 

**输出**：每个像素输出 N 个概率（N = 类别数），代表该像素属于各个类别的可能性 。

### 视觉识别 vs 大语言模型

| 维度 | 大语言模型 | 视觉识别模型 |
|------|-----------|-------------|
| 应用层 | 非常丰富（工作流、提示词、RAG） | 非常少 |
| 产品形态 | 模型能力 → 应用层丰富功能 | 模型能力 ≈ 产品功能 |
| 落地流程 | 应用层调提示词、接知识库 | 模型层+数据层定义场景、标注、训练 |
| 通用大模型 | 已出现 | 尚未出现（每场景需单独训练） |

### 行业现状

- **技术极其成熟**（20 年积累），但成本主要在数据标注和场景定义 
- **中国工厂 AI 检测普及率不高**的原因是经济账算不过来，而非技术不成熟 
- **未来期望**：出现通用视觉基座模型，成本从 50 万/场景降至 5 万/场景
