## 摘要

一种基于 Coze（扣子）工作流的自动化视频生成方案，从一句话主题出发，经过文案生成→图片生成→TTS 音频→格式重组→视频组装五步，自动产出短视频。支持两种模式：直接生成完整视频（需 API Key），或生成剪映草稿（.draft JSON）导入本地继续编辑。

## 详情

### 核心流程

```
输入：一句话主题
  ↓
① 生成故事文案（LLM，按句号分割为 N 句）
  ↓
② 循环体（每句话独立处理）：
   ├── 生成图片提示词 → 生成图片
   └── 生成音频（TTS）→ 获取时长
  ↓
③ 图片+音频+字幕 → 格式重组（Python 代码对齐时间线）
  ↓
④ 插件组装视频：创建草稿 → 添加素材 → 保存 → 生成
```



### 文案生成与分割

LLM 模型根据主题生成一小段短文，要求每句话以句号结尾，然后用文本处理节点按句号拆分为 N 个句子的数组（示例中约 10 句）。

### 循环体处理

循环节点对每个句子并行处理两路：
- **图片路径**：句子 → 扩展为英文图片提示词 → 调用模型生成图片
- **音频路径**：句子 → 火山引擎 TTS 生成语音 → 获取音频时长

**潜在问题**：图片可能因安全审核失败导致缺失，课程演示中 10 句只生成 9 张图。

### 时间线对齐

Python 代码将循环产出重组为格式化数据，图片和音频在时间线上精确对齐，段间保留约 1 秒间隔：

```
图片1：0s ────────── 4.48s    图片2：4.48s ──────── 8.216s
音频1：0s ── 3.48s ─┘间隔 1s└─ 音频2：4.48s ── 7.216s
```



### 视频组装插件

课程使用名为"简易小助手"的插件，原理是在远程服务器上操控剪映（CapCut），通过 API 调用完成素材添加和视频导出。支持 create_draft、add_image、add_audio、add_caption、save_draft、generate_video 等操作。

**草稿本质**：剪映的草稿文件是 JSON 格式，可以生成 .draft 文件（JSON）导入本地剪映继续编辑。生成草稿模式不需 API Key，免费。

### 高级变体

| 功能 | 说明 |
|------|------|
| 增加动效 | 调用海螺视频（Hailuo Video）API 让图片动起来 |
| 增加背景音乐 | 工作流中加入 BGM 素材 |
| 草稿模式 | 生成 .draft JSON → 导入本地剪映继续编辑 |
| 多种风格 | 诗意/文艺、儿童故事、神话、生活治愈、知识科普等 |



### 使用注意事项

- 直接生成视频需 API Key（按用量扣费）
- 图片生成可能因安全审核失败，建议循环体中加错误处理
- 推荐先用草稿模式（免费）熟悉流程
