## 摘要
BGE-M3（智源）是支持 100+ 语言的通用文本嵌入模型，最大上下文 8192 tokens，融合密集向量、稀疏向量和多向量检索，适用于跨语言长文档检索和高精度 RAG 场景，文件大小约 2.3G。

- **开发者**: 智源

- 适用于跨语言长文档检索、高精度 RAG
