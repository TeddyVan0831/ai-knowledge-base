## 摘要

贝叶斯推理是一种从数据中获得更清晰预测的方法，通过融入已知信息来作出更强的结论，在推理模型的 Chain-of-Thought 中提供数学基础。

## 详情

### 核心概念

- **先验概率 P(A)**：在获得新证据之前，对事件发生概率的初始估计 
- **条件概率 P(B|A)**：在已知 A 发生的情况下 B 发生的概率 
- **联合概率 P(A and B) = P(A) × P(B|A)**：两个事件同时发生的概率 
- **边缘概率 P(B)**：所有可能方式下 B 发生的总概率 

### 贝叶斯定理

P(A|B) = P(A) × P(B|A) / P(B) 

真正关心的是"如果我们知道一个人有长头发，那这个人是女性的概率是多少"——P(女人|长发)，其相反的概率我们已经知道 P(长发|女性) 。

### 电影院案例

在男卫生间排队情境下，98 名男性 + 2 名女性：
- P(男) = 0.98，P(女) = 0.02 
- P(长发|男) = 0.04，P(长发|女) = 0.5 
- P(长发) = P(男)×P(长发|男) + P(女)×P(长发|女) = 0.98×0.04 + 0.02×0.5 = 0.0492 ≈ 0.05 
- P(女|长发) = P(女)×P(长发|女)/P(长发) = 0.02×0.5/0.05 = 0.2

### 与 LLM 推理的关系

贝叶斯推理为推理模型中的 Chain-of-Thought (CoT) 提供数学基础：LLM 在生成每个 token 时，本质上是在计算 P(next_token | context)，类似于贝叶斯条件概率更新。推理模型的"长链思维"可以看作是对多个假设的贝叶斯更新过程。
