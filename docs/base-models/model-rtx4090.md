## 摘要
NVIDIA RTX 4090 是基于 Ada Lovelace 架构的消费级 GPU，拥有 16,384 个 CUDA 核心和 24GB GDDR6X 显存，是个人用户性价比最高的选择，适用于 1B-7B 模型的本地调试和原型验证。

## 详情

### 核心规格
- **架构**: Ada Lovelace 
- **CUDA 核心数**: 16,384 个 
- **显存**: 24GB GDDR6X 
- **定位**: 原型验证 

### 适用场景
- 1B-7B 本地调试 
- 消费级，无 NVLink 

### 功耗与散热
- **功耗**: 450W 
- **散热**: 标准三风扇，满载 62-70°C 
- **价格**: ~¥15,000-20,000

- **最推荐**: RTX 4090（24GB 可跑 13B-30B 量化，性价比优于 5090）
