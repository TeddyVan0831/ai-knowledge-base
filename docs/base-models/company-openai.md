## 摘要

OpenAI 是全球 AI 领导公司，以 GPT 系列大语言模型树立行业标杆，其 `/v1/chat/completions` API 协议成为事实标准。从 GPT-3 到 GPT-4o、o1、o3，持续引领闭源路线的前沿能力。

## 详情

### 核心定位

- **闭源领导者**：GPT 系列长期占据综合能力榜首
- **API 标准制定者**：Chat Completions API 被广泛兼容
- **推理模型先驱**：o1/o3 系列开创 test-time compute scaling 范式

### 代表模型

| 模型 | 发布时间 | 核心创新 |
|------|---------|---------|
| GPT-4o | 2024.05 | 原生多模态（文本+图像+音频）统一处理 |
| o1 | 2024.09 | 思维链推理模型，RL+CoT 显著提升数学/代码能力 |
| o3 | 2025.04 | 推理模型升级版，编程/科学问题大幅提升 |
